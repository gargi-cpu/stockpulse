# StockPulse Frontend

Vite + React client for the StockPulse dashboard.

## Requirements
- Node.js 18+

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and set `VITE_API_BASE_URL`.
3. `npm run dev`

## Notes
- The frontend expects the backend to be running (default: `http://localhost:4000`).
